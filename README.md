# Como funciona la Pagina

Hicimos una base de lo que va a ser el frontend. Este se compondrá por un Landing Page que mostrará lo que es la web y para que sirve. Luego si el cliente esta interesado puede crearse un usuario. Para esta entrega, el formulario de Log in está pero todavia no es funcional. 

Hicimos también una pagina principal que luego de haber ingresado su cuenta se mostrará con los perfiles de los empleados o empleadores segun el tipo de usuario. sera básicamente una tarjeta que podrá deslizarse. Por ahora está la tarjeta muy básica de 2 usuarios para mostrar el ejemplo. Si hacemos click mantenido y deslizamos veremos que cambia al siguiente usuario.

Finalmente tenemos tambien una pagina de instrucciones en el que dinamicamente se muestra la informacion de lo que podrá hacer la Pagina Web.

Tiene una navbar implementada y funcional a traves de la cual podemos navegar a las distintas paginas que creamos para esta entrega. Cosa que se vea mas o menos la idea de lo que sera la aplicacion. Si apretamos el icono volvemos a la pagina inicial.

Para esta entrega y debido al poco tiempo tenemos una muestra de lo que va a ser nuestro trabajo. Todavia el diseño no lo hemos definido bien, por eso elegimos unos colores mas al azar para que empiece a tomar forma. Pero para las siguientes entregas todo esto cambiará y será más bonito.
